## Final Polish Checklist + Final Presentation Expectations (10 minutes)
### Narrative
* Does the site guide the user through a beginning → exploration → insight?
### Interactions
* Do they reveal meaning, not distract?
### Design
* Is hierarchy clear?
* Is typography consistent?
* Does color tell a story?
### Technical
* Any console errors
* Responsive layout
* Clear file structure
* Semantic HTML
* Any Accessibility Issues
### Final Presentation Prep
* What story are you telling?
* What to demo live
* What you want your audience to feel/understand
